# ⚡ 闪电游戏

一个使用 HTML5 Canvas 和 JavaScript 开发的飞机射击游戏。

## 🎮 游戏特性

### 核心功能
- **仿真型飞机模型**：玩家和敌方飞机都采用真实的战斗机外观设计
- **四种子弹类型**：
  - 普通子弹（黄色）
  - 双发子弹（绿色，伤害1.5倍）
  - 激光子弹（紫色，伤害2倍，速度更快）
  - S型子弹（橙色，S形轨迹，伤害1.8倍）
- **10次生命系统**：玩家拥有10次生命机会
- **Boss系统**：每300分出现一个Boss，共10级关卡
- **持续射击**：按住空格键可持续发射子弹
- **爆炸效果**：敌方飞机被击毁时产生炫酷的爆炸效果
- **受伤保护**：受伤后有2秒无敌时间，带有闪烁提示

### Boss系统
- **10级关卡**：从第1级到第10级，难度递增
- **血量递增**：每级Boss血量 = 300 × 关卡数
  - 第1级：300点血量
  - 第2级：600点血量
  - ...
  - 第10级：3000点血量
- **造型升级**：关卡越高，Boss造型越坚固
  - 第3级：增加侧装甲板
  - 第5级：增加更多装甲
  - 第7级：增加重型装甲
- **多方向射击**：Boss会发射多方向子弹，关卡越高子弹越多

## 🎯 操作说明

### 基本操作
- **移动**：方向键 或 WASD
- **射击**：按住空格键持续发射子弹
- **切换子弹类型**：数字键 1/2/3/4
  - `1` - 普通子弹
  - `2` - 双发子弹
  - `3` - 激光子弹
  - `4` - S型子弹

### 游戏按钮
- **开始游戏**：开始新游戏或继续暂停的游戏
- **重置游戏**：重置所有游戏数据

## 🚀 运行方法

### 方法一：直接打开
1. 双击 `index.html` 文件
2. 游戏将在默认浏览器中打开

### 方法二：使用本地服务器（推荐）
1. 使用 Python：
   ```bash
   cd lightning_game
   python -m http.server 8080
   ```
   然后在浏览器中访问 `http://localhost:8080`

2. 使用 Node.js：
   ```bash
   cd lightning_game
   npx http-server -p 8080
   ```

### 方法三：使用 VS Code
1. 安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎨 游戏机制

### 分数系统
- 击败普通敌人：+10分
- 击败Boss：+100 × 关卡数 分

### Boss生成
- 每达到300分的倍数时，会生成对应关卡的Boss
- Boss存在时，不会生成普通敌人
- 击败Boss后，自动进入下一关卡

### 受伤机制
- 碰撞敌方飞机或敌方子弹会减少1点生命
- 受伤后有2秒无敌时间
- 无敌期间会闪烁提示
- 生命归零时游戏结束

## 🛠️ 技术栈

- **HTML5 Canvas**：游戏渲染
- **JavaScript (ES6+)**：游戏逻辑
- **CSS3**：UI样式和动画

## 📝 开发说明

游戏采用纯前端技术开发，无需任何后端服务或数据库。所有游戏逻辑都在客户端运行。

### 代码结构
- `index.html`：包含HTML结构和CSS样式
- `game.js`：包含所有游戏逻辑
  - 游戏状态管理
  - 玩家和敌人类
  - 子弹系统
  - Boss系统
  - 碰撞检测
  - 粒子效果和爆炸效果

## 🎯 游戏目标

- 尽可能获得高分
- 击败所有10个Boss
- 挑战最高关卡
